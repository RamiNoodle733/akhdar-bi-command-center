# Dashboard Screenshots

Add Power BI dashboard screenshots here after building the dashboard.

1. `01_executive_overview.png` - Main KPI cards and trend lines
2. `02_product_profitability.png` - SKU-level profit analysis
3. `03_customer_insights.png` - New vs returning, cohorts
4. `04_marketing_performance.png` - ROAS, CPC, campaign analysis

## How to Add Screenshots

1. Open Power BI Desktop
2. Connect to PostgreSQL using `powerbi_export` schema views
3. Build visualizations
4. Export each page as PNG (File > Export > Export to Image)
5. Save to this folder
